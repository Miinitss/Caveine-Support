# Caveine Support

![Profil Caveine Support](./Assets/Profil%20Caveine%20Support.png)

**Caveine Support** adalah asisten multifungsi yang dirancang untuk mempermudah tugas staf dan admin sekaligus meningkatkan pengalaman berinteraksi bagi anggota server. Bot ini menjadi jembatan utama dalam operasional harian ekosistem Server Caveine.

## 🌟 Tiga Pilar Utama

### 🎫 Ticket System
Sistem manajemen bantuan yang rapi dan efisien. Memungkinkan staf untuk menangani permintaan bantuan, klaim tiket, hingga analisis performa tim support secara terpusat.

### 🧵 Auto-Thread System
Meningkatkan diskusi yang terorganisir dengan pembuatan thread otomatis pada channel tertentu. Staf dapat melakukan klaim thread untuk memastikan setiap diskusi mendapatkan perhatian yang tepat.

### 🤝 Partnership System
Sistem manajemen mitra yang komprehensif. Mencakup pengelolaan data partner, *Partner List Interface*, hingga sinkronisasi otomatis pengumuman ke seluruh jaringan server mitra.

## ⚡ Sinergi Ekosistem & Fitur Canggih

- **Sinergi Genesis (Roles):** Bekerja sama dengan **Caveine Genesis** untuk otomatisasi pemberian role *Duta Besar (Partner)* berdasarkan database partnership.
- **Sinergi Zephyr (Promotion):** Berkolaborasi dengan **Caveine Zephyr** untuk memicu pesan promosi otomatis saat aktivitas channel mencapai ambang batas tertentu.
- **Auto Management:** Fitur **Auto Lock** channel pada jam tertentu ataupun aktivitas tertentu, **Auto Chat** untuk interaksi otomatis, dan pembersihan thread secara terjadwal.

## 🏗️ Arsitektur & Teknologi
- **Pilar Ekosistem:** Sebagai komponen "Support & Management" yang terintegrasi penuh dengan keluarga bot Caveine.
- **Database:** Menggunakan **PostgreSQL (PSQL)** di **Supabase** untuk sinkronisasi data antar bot secara *real-time*.
- **Modular Design:** Struktur kode yang terbagi ke dalam modul-modul khusus (Ticket, Partner, Autothread) untuk stabilitas tinggi.
